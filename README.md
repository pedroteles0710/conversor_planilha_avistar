# Conversor de Planilhas Avistar

## O que é?
Este sistema permite que você envie planilhas brutas (.csv) de despesas, ordens de serviço ou outros relatórios e receba automaticamente uma versão normalizada, pronta para uso em BI, análise ou integração.

## Funcionalidades
- Upload de planilhas brutas (.csv)
- Normalização automática dos dados (compatível com diferentes formatos)
- Download do resultado em CSV ou Excel (.xlsx)
- Pré-visualização dos dados normalizados antes do download
- Visualização dos dados em formato JSON

## Como usar
1. **Acesse o sistema**
   - Local: http://localhost:5000
   - Online: [Seu link Render]
2. **Envie sua planilha bruta**
   - Clique em "Escolher arquivo" e selecione o arquivo .csv
   - Clique em "Enviar e Normalizar"
3. **Veja a pré-visualização**
   - Após o processamento, uma tabela com as primeiras linhas será exibida
4. **Baixe o resultado**
   - Use os links para baixar o arquivo normalizado em CSV ou Excel
   - Ou visualize em JSON

## Observações
- O sistema aceita planilhas de diferentes layouts (despesas, ordens de serviço, etc.)
- O arquivo baixado terá data e hora no nome para facilitar o controle
- Para dúvidas ou problemas, entre em contato com o responsável pelo sistema
